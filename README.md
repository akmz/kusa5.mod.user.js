## これ

kusa5.user.js の改造版です。  
以下の変更点を含みます。

#### 全般

- ファイルタイプを判定して、MP4動画のみ実行
- 既定では画面上部にプレーヤーを移動しない
- オプションの追加（察してくれ）

### API

- コミュニティ動画に対応

#### プレーヤー

- レスポンシブ対応
- 色・サイズのコマンドに対応
- ループ再生対応
- 音量の無段階調節
- 一般会員はバッファ済みの範囲でシーク可能（プレミアム会員は無制限）

## これ

kusa5.user.js の改造版です。  
以下の変更点を含みます。

#### 全般

- ファイルタイプを判定して、MP4動画のみ実行
- 既定では画面上部にプレーヤーを移動しない
- オプションの追加（察してくれ）

#### プレーヤー

- レスポンシブ対応
- 色付きコメント対応
- 音量の無段階調節
- プレミアム会員のみシーク可能（一般会員はシーク不可）
